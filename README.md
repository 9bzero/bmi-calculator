# BMI Calculator

Calculate your Body Mass Index with a visual gauge, category color coding, and health tips.

## BMI Categories

| BMI | Category |
|---|---|
| < 18.5 | Underweight |
| 18.5 – 24.9 | Normal weight |
| 25.0 – 29.9 | Overweight |
| ≥ 30.0 | Obese |

## Features

- Metric (kg/cm) and imperial (lb/ft) input
- Animated gauge needle pointing to your category
- Color-coded result (blue → green → yellow → red)
- Ideal weight range for your height
- Basic health tips per category
