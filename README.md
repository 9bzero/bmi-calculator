# bmi-calculator

Calculate BMI and see where you land on the scale.

## Categories

| BMI | Category |
|---|---|
| < 18.5 | Underweight |
| 18.5 – 24.9 | Normal |
| 25.0 – 29.9 | Overweight |
| ≥ 30.0 | Obese |

Metric (kg/cm) and imperial (lb/in) input. Animated gauge needle. Shows the healthy weight range for your height.

## Run

```bash
npm install && npm run dev
```
